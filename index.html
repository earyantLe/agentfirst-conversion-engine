🤖 Agent 优先网站转换引擎

Agent-First Browser Automation

功能特性

  • 原子化 Action 体系 - 6 大类 20+ 原子操作（Navigation、Snapshot、Interaction 等）
  • 统一接口规范 - 每个 Action 有明确的输入/输出/状态
  • 工作流编排引擎 - 支持条件分支、并行执行、错误恢复
  • 确认/回滚机制 - 敏感操作需确认，失败自动回滚

安装

npm install agentfirst-conversion-engine

快速开始

import { ConversionEngine } from 'agentfirst-conversion-engine';

const engine = new ConversionEngine();

const workflow = await engine.analyzeUrl('https://example.com');
const actions = await engine.generateWorkflow(workflow);
await engine.execute(actions);
